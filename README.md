# Модель предсказания спроса в сервисе велопроката
## Описание

Ключевые слова: регрессия, kNN, DecisionTree


## Требования

- **Python**: 3.11 или выше
- **ОС**: Linux, macOS или Windows (с WSL)

## Установка и запуск

#### Для Fish shell:

```fish
fish start.fish
```

#### Для Bash/Zsh:

```bash
./start.sh
```
